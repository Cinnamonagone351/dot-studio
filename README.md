## About dot-studio

dot-studio is a figma-style AI orchestration tool. It helps you manage AI workflows using an easy visual interface. You do not need programming skills to use it. The software works with various AI agents and models, simplifying tasks like coordinating language models or managing data flows.

This tool suits users who want a clear way to control AI processes without writing code. It runs on Windows and offers a smooth, intuitive environment to build AI systems step-by-step.

Key parts of dot-studio:

- Visual editor that feels like design apps you know.
- Connects to different AI services and models.
- Lets you create tasks by dragging and dropping elements.
- Runs large language models (LLMs) and AI agents easily.
- Saves and shares your work with others.

This project fits topics like agentic AI, AI orchestration, and handling large language models.

## ⚙️ Running dot-studio for the First Time

After installation, launch dot-studio from your Start menu or desktop shortcut.

When you open it the first time, a welcome screen will appear:

- You can create a new AI orchestration project.
- Or open an existing project if you have one saved.

The interface looks like a design tool with panels for components, workspace, and properties.

To start a new project:

1. Click “New Project”.
2. Name your project and choose where to save it.
3. Use the visual toolbox on the left to drag AI components into the main workspace.
4. Connect these components by drawing lines.
5. Each component represents an AI task or service.
6. Click on a component to configure its details on the right panel.
7. Use the play button to test your setup.

dot-studio runs in real time and shows results or errors promptly.
